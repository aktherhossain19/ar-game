DATA STRUCTURES IN THIS GAME

📦 QUEUE — Target Spawn System

প্রতিটা target spawn হওয়ার আগে একটা spawnQueue তে বসে। FIFO rule মেনে চলে — প্রথমে যে target queue এ ঢুকেছে, সেটা আগে spawn হয়।

Wave বাড়লে queue তে বেশি targets push হয়। dequeue() call হলে পরবর্তী target বের হয়ে AR scene এ add হয়। Bottom-left এ যে boxes দেখছো সেটা live queue visualization।

📚 STACK — Score History

প্রতিটা shot (hit বা miss) একটা scoreStack তে push হয়। LIFO — সবচেয়ে নতুন entry সবার উপরে।

Bottom-right panel এ stack এর top 5 entry দেখাচ্ছে। peek() দিয়ে last action check করা হয়, combo multiplier calculate করতে।
